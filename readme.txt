== Description ==

This plugin integrates Beaver Builder plugin with the theme Dynamik Website Builder, and create two page templates, one full width and the other one a landing page.

Thanks to https://hbt.io for the idea, and Junior Atoms (http://cobaltapps.com/forum/member/18-junior-atoms) for the codes.

**NOTE: This plugin require to have installed the theme [Dynamik Website Builder](http://cobaltapps.com/downloads/dynamik-website-builder) which required [Genesis](http://www.studiopress.com), and the plugin [Beaver Builder](https://www.wpbeaverbuilder.com). The plugin won't be activated unless you have satisfied all the requirements.**

== Installation ==

To install this plugin just follow the standard procedure.

or

1. Upload `dwbandddintegration.zip` to the `/wp-content/plugins/` directory
2. Unzip it
2. Activate the plugin through the 'Plugins' menu in WordPress
